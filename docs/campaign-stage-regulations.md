# 塔ダンジョン レギュレーション一覧

MonoKnight の現行メインコンテンツは、カードでマス目を移動して塔を登る **出口到達型ダンジョン**である。ライトユーザーは低難度ダンジョンで周回成長しながら登り、高難度ダンジョンではローグライク式に毎回初期状態から挑む。実装の一次情報は `Game/DungeonDefinition.swift` とし、旧目的地制キャンペーンの定義コードは削除済みで現行仕様として扱わない。

## 共通仕様

- 塔攻略の基本クリア条件は `completionRule = .dungeonExit(exitPoint:)` とする。
- 各フロアは入口、出口、HP、残り手数、敵、床ギミックを持つ。
- 塔ダンジョン本編は通常の手札/NEXT/山札ドローではなく、基本移動と拾得/報酬カードの回数制インベントリを使う。
- 所持カードは通常カード9種類まで。基本移動が許可されたフロアでは10枠目を消費されない基本移動カードとして固定表示し、同じカードは種類枠を増やさず、残り使用回数として積む。
- 目的地補充は塔ダンジョン本編では使わない。
- 敵はプレイヤーのカード使用後にターン制で行動する。
- ダンジョン処理順は `床ギミック → 出口判定 → 敵ターン → 失敗判定` とする。
- 低難度ダンジョンでは永続成長を持ち込み可能にし、高難度ダンジョンでは永続成長を無効化または記録/外見のみに限定する。

## ダンジョン構成

| 区分 | テーマ | 成長 | 主なギミック |
|----|--------|------|--------------|
| チュートリアル | 基礎塔 | 成長なし | 出口到達、HP、手数、番兵、ひび割れ床 |
| 標準 | 成長塔 | 永続成長あり | 巡回兵、追跡兵、メテオ兵、暗闇、施錠階段、ワープ床、見える罠、固定岩/柱、床割れ、複合ギミック |
| 中難度 | 麻痺/誘導塔、高難度前の複合塔 | 一部持ち込みまたは一時報酬中心 | 麻痺、押し出し、報酬選択、複合ギミック |
| 高難度 | 崩れる塔、視界制限塔、複合ランダム塔 | 毎回初期状態 | 床割れ、敵増加、視界制限、ランダム階層 |

今後の開発順と判断基準は [`dungeon-development-roadmap.md`](dungeon-development-roadmap.md) を正本とし、本書は実装済み/実装予定の塔ごとの規定を記録する。

高難度塔の v1 プロトタイプは、崩れる床、視界制限、敵増加、ランダム階層の組み合わせまでに留める。カード削除/強化、罠、ワープの既存要素は使っても、成長塔の永続成長を前提にしない。

## 初期実装の塔

- `DungeonLibrary.shared` は当面 `基礎塔`、`成長塔`、`試練塔` の3本だけを提供する。
- 基礎塔はチュートリアル用の 9×9 / 6 フロア構成とし、出口到達、見張り、ひび割れ床、鍵と施錠出口、見える罠、ワープ床と短い巡回兵を順に確認する。永続成長ポイント、スキルツリー効果の対象外にする。
- 成長塔は 9×9 / 50 フロア構成とし、旧 `巡回塔`、`鍵扉塔`、`ワープ塔`、`罠塔` の内容を独立塔としては表示せず、段階的に統合する本編塔にする。
- 試練塔は 9×9 / 無限階の `.roguelike` 塔とし、毎回初期状態から始め、永続成長ポイント、スキルツリー効果、開始カード、成長補正、遺物の対象外にする。ラン中だけ HP、拾得/報酬カード、累計手数を持ち越す。
- 巡回兵がいるフロアでは巡回路全体を薄い灰色のレールとして常時表示し、巡回兵の次に進む方向は敵アイコン内に表示する。追跡兵の次に進むマスへの矢印は盤面に常時表示し、回転見張りの右回り/左回りは敵アイコンの瞳孔部分に表示する。
- メテオ兵がいるフロアでは、次の敵ターンに危険になる着弾予告マスを表示し、そのマスに残った場合は1ダメージを受ける。
- 暗闇フロアでは、プレイヤー周囲1マスと階段、見張り系レーザー、メテオ着弾予告だけを常時表示し、敵本体、鍵、拾得カード、罠、宝箱、床効果、巡回レールは視界内だけ表示する。
- 塔モードの四角い枠は移動可能マスだけに使い、床落ちカードや床ギミックは枠なしのマーカーとして表示する。
- タイトルの通常プレイ入口は塔選択 UI を開き、選んだ塔を1Fから連続で開始する。
- 基礎塔と成長塔では上下左右1マスの基本移動を常時許可する。基本移動はカードを消費せず、手数、敵ターン、床ギミック、出口判定、疲労判定の対象になる。
- 基礎塔と成長塔はカードなしで始まり、基本移動だけでも出口へ届くが、床落ちカードや報酬カードを使うと 2〜4 手程度短縮できる。
- 成長塔1-8Fは、罠、敵、床割れを増やして難しくするのではなく、拾得カードを各5枚前後へ増やして盤面の選択肢と易しさを出す。
- 成長塔は全50Fに2〜4個程度の固定岩/柱を置く。岩/柱は `impassableTilePoints` の通行不可マスで、鍵を取っても消えず、spawn / exit / 鍵 / 拾得カード / 敵初期位置 / 罠 / ひび割れ床 / ワープ床とは重ねない。
- 追跡兵は成長塔7F以降に少量だけ配置し、プレイヤーの1手後に通行可能な最短経路で1マス近づく。表示する危険範囲は上下左右1マスで、敵本体マスは敵マーカーとして分け、待ちすぎや寄り道しすぎでHPを削られる圧を作る。
- メテオ兵は成長塔17F以降に少量だけ配置し、制御ランダムな着弾予告マスで安全な待機マスをずらす。
- 暗闇は成長塔17F以降の数フロアだけに置き、メテオや見張り系と組み合わせて終盤の情報制限として扱う。
- 巡回兵は成長塔9F以降で6〜8マス程度の巡回範囲へ段階的に広げる。巡回パスは上下左右1マスずつ連続し、spawn / exit / 鍵 / 拾得カード / ワープ床 / 固定岩柱 / 罠 / ひび割れ床 / 他敵とは重ねない。
- 1F は基本移動と2マス移動の短縮、2F は見張りの縦射線を避ける迂回、3F はひび割れ床列を迂回するかカードで抜ける判断、4F は鍵取得で施錠出口が開くこと、5F は見える罠を迂回するかHPを支払って近道する判断、6F はワープ床と短い巡回兵を主役にする。
- 3F は中央のひび割れ床列を直進すると番兵の危険範囲で HP を失う。迂回または2F報酬や床落ちカードを使うと損失を抑えて短縮できる。
- フロア間では同一区間内だけ HP と所持カードの残り使用回数を引き継ぎ、手数、敵、床状態、床落ちカード配置は各フロア開始時にリセットする。
- フロア内カード拾得は、そのマスを踏むと追加手数なしで取得する。拾得カードは1回使い切りで、クリア時に未使用分が残っていれば次フロアへそのまま持ち越す。
- 上下左右1マスカードと上下/左右選択キングは廃止済み。上下左右1マスは基本移動のみが担当する。
- 次フロアがあるクリア後は、既存移動カードの 3択報酬から1枚を手札に追加するか、所持報酬カードの使用回数 +1 を選べる。所持報酬カードは報酬を消費せず、報酬選択前に何枚でも手札から外せる。手札に追加した移動カードは通常2回使用可能なラン中報酬として次フロアへ持ち越す。試練塔は無限進行のため、毎階クリア後に次階へ進む報酬導線を維持する。
- 1F 報酬は `右2`、`上2`、`右桂 (選択)`、2F 報酬は `右連続`、`右2`、`右桂 (選択)` とする。
- 6F クリア後は報酬を出さず、基礎塔クリアとして完了する。
- 成長塔は1-10F、11-20F、21-30F、31-40F、41-50Fの5区間で構成する。10F/20F/30F/40Fクリアで次区間開始を解放し、通常報酬を選んで同じランのまま次区間へ進める。解放済み区間から新規開始する場合はHPと報酬カードを初期化する。連続進行では前フロアの階段マスを次フロアの開始マスにし、階段位置は複数パターンへ分散して毎周同じ配置感になりすぎないようにする。
- 成長塔の 1-49F の報酬3択は、次フロアの安全ルート、短縮ルート、拾得カードや鍵/罠/宝箱への寄り道のいずれかへ効く候補にする。10F/20F/30F/40F は成長区切りだが報酬を選んで次区間へ進める。50F は最終階として報酬を出さない。
- 成長塔では5F刻みの初回クリアで成長ポイントを各1得る。同じ区切りでは重複付与しない。
- 試練塔は挑戦開始時に生成 seed を作り、seed と階数から各階を決定論的に生成する。同じ seed と同じ階数では中断復帰後も同じ配置になり、違う seed または階数では別配置になる。移動不能マスだけで入口から出口までの上下左右経路が詰む盤面は生成せず、入口、出口、鍵、敵、罠、拾得カード、通行不可、ワープを重ねない。1Fから数階は軽めにし、以後は階数に応じて罠、敵、床割れ、ワープ、暗闇、状態異常床を段階的に増やす。階層表示は `試練塔 12F` のように分母なしで表示し、ローカル最高到達階を塔選択とリザルトに表示する。
- 失敗時の再挑戦は、成長塔では現在の区間開始階、基礎塔と試練塔では1Fから始める。試練塔は再挑戦ごとに新しい生成 seed の新規ランにする。
- 永続成長、報酬カードの強化、報酬を消費しない持ち越し整理は実装済み。成長は3枝型スキルツリーとし、開始カード、報酬候補、報酬使用回数、罠/床崩落ダメージ軽減に限定する。麻痺罠は見える床ギミックとして扱い、補助/攻撃/押し出し系報酬は次フェーズ以降で検討する。

## QA 観点

- 出口到達でクリアし、目的地数に依存しないこと。
- 塔開始時に手札/NEXT/山札補充が行われず、所持カードが空であること。
- 基本移動で所持カードが変わらず、手数とダンジョンターンだけ進むこと。
- 拾得カードは使用で回数が減り、クリア時の未使用分が次フロアへそのまま持ち越されること。
- 基礎塔と成長塔の各フロアで、基本移動だけの代表ルートとカード短縮の代表ルートが成立すること。
- 成長塔の鍵フロアで、鍵マスを踏むまで階段が施錠され、取得後に鍵アイコンが消えて階段へ到達できること。序盤の鍵学習では可変式の障害物扉を使わず、固定岩/柱を含めても基本移動だけの代表ルートと鍵/カード短縮の代表ルートが成立すること。
- 暗闇フロアでも開始地点から階段、鍵ありなら開始地点から鍵と鍵から階段までの代表ルートが成立すること。
- HP 0 で失敗すること。残り手数 0 後は即失敗ではなく、上限超過1手目と以後3手ごとに疲労ダメージを受けること。代表ルートには余裕を持たせ、迷走防止の疲労開始ラインとして機能すること。
- 出口到達した手は、敵ターンや疲労判定より先にクリアすること。
- 敵の危険範囲表示は敵本体マスを含まず、攻撃されるマスだけを示すこと。敵本体マスは入ると撃破でき、その手の敵ターンでは攻撃しないこと。
- 番兵、巡回兵、見張り、回転見張り、追跡兵、メテオ兵の挙動がターンごとに再現可能であること。
- ひび割れ床が 1 回目でひび割れ、2 回目で崩落穴として残り、入るたびに HP 1 ダメージと下階同座標への落下を発生させること。落下先の同座標が無傷ならひび割れて停止し、既にひび割れている場合だけ連鎖落下すること。成長塔では、区切り階や最終階など通常遷移できる次階がない階にひび割れ床を置かないこと。
- 区間内の報酬カードが所持カードとして次フロアへ反映され、区間をまたぐ開始では破棄されること。
- 成長塔10Fクリア後に11F開始が解放されること。
- 成長塔では永続成長が反映され、基礎塔と試練塔では無効化されること。
- タイトル、設定、リザルトから旧キャンペーン、Daily、ハイスコア、旧ランキング文脈が出ないこと。
