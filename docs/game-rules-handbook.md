# MonoKnight ゲームルールハンドブック

本書は MonoKnight に実装されているゲームルールの詳細 Source of Truth とする。
プロダクト要件は [`product-spec.md`](product-spec.md) を併用し、旧仕様アーカイブの [`agents_legacy.md`](agents_legacy.md) は現行判断に使わない。
現行メインコンテンツは塔攻略のみであり、旧目的地制キャンペーン、Daily、ハイスコア、旧標準/クラシカル/Target Lab は現行仕様として扱わない。

## 1. 盤面・座標・初期配置

- 盤面は `BoardGeometry` が提供する N×N グリッド。塔フロアは `DungeonFloorDefinition` ごとの盤面サイズを使う。
- 基礎塔、成長塔、試練塔は原則 9×9 とする。
- 座標系は左下が原点 `(0,0)`。右方向が `+x`、上方向が `+y`。UI 表示では下段が y=0。
- 塔フロアの開始位置は `DungeonFloorDefinition.spawnPoint` で定義する。同一区間内では前フロアの階段位置を次フロアの開始位置にできる。
- 盤面の四角い枠は、このターンに実際に移動できるマスだけを示す。床落ちカード、危険範囲、ダメージ罠、ひび割れ/崩落床は枠ではなく塗りや小マーカーで表現する。レイ型などの通過塗りが敵の危険範囲と重なる場合は、ダメージを示す赤塗りを優先し、終点の移動可能枠だけ残す。

## 2. カードと所持品

- 塔ダンジョン本編では通常の手札/NEXT/山札補充を使わず、基本移動と所持カードの回数制インベントリで進む。
- `DungeonFloorDefinition.cardPickups` のマスを踏むと、追加手数なしで移動/補助カードを取得する。拾得カードは1回使い切りで、クリア時に未使用分が残っていれば次フロアへそのまま持ち越す。床拾得には遺物を混ぜない。
- 成長塔の `DungeonFloorDefinition.relicPickups` の宝箱マスを踏むと、追加手数なしで遺物を取得する。遺物はカード枠とは別管理で、その挑戦中だけ効果が続く。同じ通常遺物/呪い遺物は1回の挑戦中に重複して出ない。取得結果は移動演出後に軽い詳細表示を挟み、確認タップで同じフロアの操作へ戻る。
- 床で拾ったカードとフロアクリア後に選んだカードはどちらも同じ所持カードとして扱い、残り使用回数を `DungeonRunState.rewardInventoryEntries` に保持して連続進行するフロア間で持ち越す。
- 所持カードは通常カード9種類までにし、補助カードも通常所持枠を1種類として数える。基本移動が許可された塔では10枠目を消費されない基本移動カードとして固定表示する。同じカードは種類枠を増やさず残り使用回数として積み、使用時は拾得分から先に消費する。
- 9種類満杯の状態で未所持の床落ちカードを踏んだ場合は、拾ったカードと所持中の9種類から捨てるカードを選ぶ。拾ったカードを選ぶと取得せず盤面から消し、所持カードを選ぶとその種類の残り回数をすべて捨てて拾ったカードを取得する。同種カードを踏んだ場合は選択を挟まず自動で回数を積む。この取捨選択では追加手数やペナルティは発生しない。
- 報酬は次フロアがあるクリア後に選ぶ。成長塔10Fは成長区切りだが通常報酬を選んで11Fへ進める。試練塔は無限進行のため毎階クリア後に報酬を選んで次階へ進める。新しい移動/補助カードを手札に追加するか、低確率の未所持遺物をカード枠なしで取得するか、所持カードを強化/整理する。移動/補助/遺物は同じ候補枠に並び、通常は最大3択、広い見立て、`勝利の旗`、`割れた羅針盤` の補正込みで最大4択にする。試練塔の報酬候補は成長スキル補正を受けず、生成階数に応じた最大3択として扱う。報酬で遺物を選んだ場合は詳細表示を確認してから次フロアへ進む。
- 手札に追加した移動カードは通常2回使用可能にする。成長スキル「カード温存」取得後は対象の使用回数を3回にする。補助カードは強力な1回報酬として扱い、カード温存の回数増加対象にしない。
- `補給` は移動せず1手使って自身を消費し、通常カード枠が9種類まで埋まるように、塔用移動カード全体から未所持の移動カードをランダムに補給する。補給後はその場でターンが経過し、溶岩滞在ダメージと敵ターンが発生する。上下左右1マスカードと上下/左右選択キングは廃止済みで、上下左右1マスは10枠目の基本移動だけが担当する。補給で追加されたカードは現在フロア限りの1回カードで、クリア時に未使用分が残っていれば次フロアへそのまま持ち越す。
- 呪文系補助カード `消滅の呪文` は、手札で選んでから盤面上の敵を1体選び、移動せず1手使って自身を消費し、その敵だけを消滅させる。効果解決後はその場でターンが経過し、溶岩滞在ダメージと敵ターンが発生する。カード拾得は発生しない。敵がいないフロアでは使用できず、敵以外を選んだ場合は消費しない。
- 呪文系補助カード `全滅の呪文` は、移動せず1手使って自身を消費し、現在フロアの敵をすべて消滅させる。効果解決後はその場でターンが経過し、溶岩滞在ダメージと敵ターンが発生する。カード拾得は発生しない。敵がいないフロアでは使用できない。
- 呪文系補助カード `凍結の呪文` は、移動せず1手使い、使用直後から合計3回分の敵ターンを止める。凍結中は危険範囲、着弾予告、次移動プレビューを表示せず、敵の攻撃範囲やメテオ予告に入ってもダメージを受けない。
- 呪文系補助カード `障壁の呪文` は、移動せず1手使い、使用直後から合計3回分の行動後処理でHPダメージだけを無効化する。敵、メテオ、ダメージ罠、溶岩、毒、崩落穴のHPダメージは受けないが、敵の移動/回転/メテオ更新、危険範囲表示、床破壊、落下、足枷、手札喪失などの非HP効果は通常どおり発生する。
- 呪文系補助カード `ダークネスの呪文` は、移動せず1手使い、その階にいる間だけ見張りと回転見張りのレーザー攻撃を封じる。見張り系の射線危険と射線ダメージは消えるが、敵本体、回転見張りの回転、番兵/巡回兵/追跡兵/メテオ兵、罠や床効果は通常どおり残る。見張り系がいない階や、すでにダークネス中の階では使えない。
- 呪文系補助カード `レール破壊の呪文` は、移動せず1手使い、その階にいる間だけ巡回兵のレール移動を封じる。巡回レールと次移動プレビューは消え、巡回兵は敵ターンで移動しないが、敵本体と隣接危険/攻撃は通常どおり残る。巡回兵がいない階や、すでにレール破壊中の階では使えない。
- `解毒薬` は、毒状態の時だけ移動せず1手使って自身を消費し、毒の残りダメージ回数と次ダメージまでの行動数を0にする。毒状態でない時は使えない。
- `万能薬` は、毒・足枷・幻惑状態のいずれかの時だけ移動せず1手使って自身を消費し、対象の状態を解除する。使用開始時に足枷中なら、その使用自体は2手分として数えるが、解除後の敵ターンは通常どおり最大1フェーズになる。麻痺罠の即時休み、凍結の呪文、障壁の呪文は解除しない。
- 現行補助カードは `補給`、`消滅の呪文`、`全滅の呪文`、`凍結の呪文`、`障壁の呪文`、`ダークネスの呪文`、`レール破壊の呪文`、`解毒薬`、`万能薬` とする。`NEXT更新`、`入替`、`導き` は退役カードとして扱い、塔攻略には出さない。
- `全域ワープ` と `固定ワープ` は退役カードとして扱う。ワープ要素は `TileEffect.warp` のワープ床として残し、カード報酬や床落ちカードには出さない。
- `DungeonRules.allowsBasicOrthogonalMove` が true のフロアでは、10枠目の基本移動カードから上下左右1マスへ基本移動できる。基本移動はカードを消費せず、手数を1増やし、敵ターン、床ギミック、出口判定、疲労判定の対象になる。ガイドモードがオンの場合は従来どおり盤面に表示された基本移動候補を直接タップできる。ガイドモードがオフの場合は、基本移動カードまたは通常カードを先に選び、表示された候補マスをタップして確定する。

## 3. クリア・失敗・処理順

- 各フロアは `GameMode.CompletionRule.dungeonExit(exitPoint:)` を使い、指定出口/階段マスへ到達するとクリアする。
- レイ型など途中マスも踏むカードでは、移動途中で解錠済みの出口/階段へ入った時点で移動を止めてクリアする。
- ダンジョン失敗条件は `DungeonFailureRule` で管理する。現行の基本失敗条件は HP 0 とする。手数上限は即失敗ではなく疲労開始ラインとして扱い、上限超過1手目、その後は超過4手目、7手目、10手目...で防御/障壁では防げない疲労ダメージを1受ける。残り手数が0になると、UIの手数ゲージ領域は3つの丸による疲労インジケーターへ切り替わり、安全な手数消費中ではないことを示す。
- レイ型など途中マスも踏むカードでは、通過マスごとに `踏破 → 敵踏み潰し → 敵攻撃範囲ダメージ → 罠/床崩落 → 拾得カード → 鍵/出口判定 → 麻痺罠` を処理し、表示もその順で一歩ずつ反映する。HP 0、床崩落、出口到達、麻痺罠などで移動が止まる場合は、その先のマスは踏まない。
- プレイヤー移動が止まった後の処理順は `出口判定 → 敵ターン → 疲労判定 → 失敗判定`。出口へ着いた手は、疲労や敵ターンより先にクリアとして扱う。麻痺罠で止まった場合は1回休みとして表示し、通常の敵ターンを最大2回連続で解決する。凍結の呪文が有効な敵ターンは、敵の移動、回転、攻撃、メテオ着弾を丸ごとスキップして残り凍結回数を1減らす。障壁の呪文が有効な行動後処理では、敵ターンや溶岩滞在処理を通常どおり解決したあと、HPダメージだけを0にして残り障壁回数を1減らす。疲労ダメージは障壁では防げない。
- フロア間では同一区間内だけ HP、所持カードの残り使用回数、成長塔の所持遺物を引き継ぎ、手数、敵、床状態、床落ちカード配置は各フロア開始時にリセットする。試練塔ではラン中だけ HP、拾得/報酬カード、累計手数を持ち越し、永続成長、開始カード、成長補正、遺物は持ち込まない。
- 失敗時の再挑戦は、成長塔では現在の区間開始階、基礎塔と試練塔では1Fから始める。試練塔は再挑戦ごとに新しい生成 seed の新規ランにする。

## 4. 塔の種類

- `DungeonLibrary.shared` は当面 `基礎塔`、`成長塔`、`試練塔` を提供する。
- 基礎塔はチュートリアル用の 9×9 / 6 フロア構成とし、出口到達、HP、手数、見張り、番兵、ひび割れ床、鍵、見える罠、ワープ床、短い巡回兵を順に確認する。成長塔はロックしないが、基礎塔未クリアで成長塔を初回開始する時だけ基礎塔を勧める。永続成長ポイント、スキルツリー効果の対象外にする。
- 成長塔は 9×9 / 50 フロアの `.growth` 本編塔とする。1-10F、11-20F、21-30F、31-40F、41-50F の5区間で構成し、10F/20F/30F/40Fクリア後は次区間から開始できる。
- 成長塔では5F刻みの初回クリアで成長ポイントを各1得る。同じ区切りでは重複付与しない。
- 試練塔は 9×9 / 無限階の `.roguelike` 塔とし、永続成長ポイント、スキルツリー効果、開始カード、成長補正、遺物、塔ごとの成長獲得表示の対象外にする。挑戦開始時に生成 seed を作り、seed と階数から各階を決定論的に生成する。同じ seed と同じ階数では中断復帰後も同じ配置になり、違う seed または階数では別配置になる。階層表示は `試練塔 12F` のように分母なしで表示し、ローカル最高到達階を保存する。
- タイトルの通常プレイ導線は塔選択だけを表示する。旧キャンペーン、Daily、ハイスコア、旧ランキング画面は通常導線に出さない。

## 5. 成長塔

- 成長塔は巡回、鍵、見える罠、ワープ床、敵警戒、回転見張り、追跡兵、メテオ兵、暗闇、固定の岩/柱、床割れを段階的に複合する。
- 暗闇フロアでは、プレイヤー現在地を含む周囲1マス、階段、見張り/回転見張りのレーザー危険マス、メテオ着弾予告マスだけを常時表示する。敵本体、鍵、拾得カード、宝箱、罠、床効果、巡回レールなどは視界内だけ表示し、判定や取得、被弾、敵行動は通常フロアと同じまま扱う。
- 岩/柱は1フロア2〜4個程度の固定障害物で、鍵を取っても消えない。
- 1-8Fは難しいギミックを足す代わりに拾得カードを各5枚前後へ増やし、拾うほど短縮、安全ルート、鍵やカード回収への寄り道が楽になる低難度区間として扱う。
- 成長塔ではラン開始時に `DungeonRunState.cardVariationSeed` を決め、同じラン中は固定したまま、拾得カードの種類/位置/数、敵初期位置/向き/巡回路、鍵位置、ワープ位置、階段接続、岩/柱、床ギミック、宝箱位置、クリア後報酬候補を階ごとに解決する。敵種、敵数、鍵の有無、ワープペア数、階テーマ、手数上限は固定テンプレートを維持する。階段は同一区間内で近傍候補へ揺れ、前階の階段位置が次階の開始位置になる。1F/11F/21F/31F/41Fの区間開始地点は固定する。拾得カード数は既存数を基準に -1 / 0 / +1 程度で揺れ、1-8F は4〜6枚、9F以降は最低3枚を保つ。岩/柱は階層帯ごとの軽い増減に留め、鍵あり階では開始地点から鍵、鍵から階段まで基本移動だけの代表導線を残す。ワープは必須ルートではなく選択肢として扱い、ワープなしでも階段へ到達できる。カード種別は 1-5F、6-10F、11-15F、16-20F、21-30F、31-40F、41-50F の階層帯と床拾得/クリア報酬ごとの重み付きプールから抽選する。解毒薬は6F以降、万能薬、ダークネスの呪文、レール破壊の呪文は11F以降の補助枠に低確率で混ぜる。床拾得は移動約90%/補助約10%/遺物0%。クリア報酬は通常時に移動約89%/補助約10%/遺物約1%で、クリア手数が有効手数上限の70%以下なら 77%/20%/3%、50%以下なら 65%/30%/5% へ寄せる。同じ seed と同じクリア手数では同じ候補になる。
- 成長塔には数階に1個の宝箱を置く。宝箱は踏むとランダムな未所持遺物を即取得し、取得済み宝箱は同じラン中に盤面から消える。宝箱は spawn / exit / 鍵 / 敵 / 罠 / 岩柱 / ワープ / 拾得カードとは重ねない。後半の一部では、上階の崩落穴からだけ入れる小部屋に宝箱を置く。小部屋には `returnWarp` の帰還ワープを置き、宝箱取得後に通常エリアへ戻れる任意の寄り道報酬として扱う。通常宝箱で未所持通常遺物が尽きた場合は重複付与せず、HP +1 の小補填だけ行って宝箱を消す。通常遺物、呪い遺物、ミミック、パンドラ、補填結果はいずれも取得結果として詳細表示する。
- 通常遺物にはコモン、レア、レジェンドの希少度を持たせる。通常宝箱の通常遺物抽選は コモン75 / レア22 / レジェンド3、怪しい宝箱・軽量は コモン55 / レア35 / レジェンド10、怪しい宝箱・深層は コモン40 / レア42 / レジェンド18 の重みを使う。クリア報酬のレリック枠は通常時 コモン70 / レア25 / レジェンド5、70%以内クリアで 60 / 32 / 8、50%以内クリアで 45 / 40 / 15 に寄せる。`曇った鏡` がある場合は手早いクリアによる希少度補正を無効化する。
- 宝箱は通常宝箱と怪しい宝箱に分ける。通常宝箱は通常遺物だけを出す。怪しい宝箱は盤面表示と VoiceOver 名を変え、軽量版は通常遺物75% / 呪い遺物15% / ミミック7% / パンドラ3%、深層版は通常遺物60% / 呪い遺物25% / ミミック10% / パンドラ5%で解決する。通常遺物、呪い遺物、パンドラの候補が成立しない場合はその結果を抽選テーブルから外し、残りの重みで再正規化する。12Fは軽量版、16F/19Fは深層版にする。
- 遺物と呪い遺物はゲームオーバー、区間開始からの再挑戦、新規開始、最終クリアで失われる。中断復帰では `DungeonRunResumeSnapshot` に所持遺物、所持呪い遺物、取得済み宝箱を保存して保持する。基礎塔と試練塔には宝箱と遺物を入れない。
- 通常遺物は48種とする。追加遺物は `木彫りの護符`、`銅の砂時計`、`旅の保存食`、`小さなランタン`、`鈍い針`、`継ぎ接ぎロープ`、`野戦医療箱`、`斥候の羅針盤`、`早業の鞘`、`清めの護符`、`不死鳥の羽根`、`賢者の写本`、`罠踏みの靴底`、`残り火の外套`、`見張りの単眼鏡`、`レール守りの護符`、`追跡避けの囮`、`解毒石`、`星除けの傘`、`落下止めの錨`、`折りたたみ地図`、`幻の切符`、`焚き火の熾火`、`商人の天秤`。通常遺物は直接的なマイナス効果を持たず、有限回数を持つものは一時レリック、回数なしで挑戦中続くものは永続レリックとして表示する。通常宝箱、怪しい宝箱、パンドラ、クリア報酬では所持済み通常遺物を候補から除外する。
- `割れた盾` は次に受けるダメージを1回だけ1軽減する。
- `重い王冠` はクリア後に新しく得る報酬カードの使用回数を1増やす。
- `灯る心臓` は取得時に HP を2増やす。
- `古い地図` は宝箱取得後、そのフロアの未取得拾得カードを盤面上で見つけやすくする。
- `黒い羽根` は崩落穴による落下を1回だけ無効化する。
- `欠けた砂時計` は各フロアの手数上限を3増やす。

- 怪しい宝箱の `ミミック` は即時に HP を2減らし、HP0ならその場で失敗する。`パンドラの箱` は未所持の通常遺物1つと未所持の呪い遺物1つを同時に得る。
- 呪い遺物は28種とする。怪しい宝箱とパンドラでは所持済み呪い遺物を候補から除外する。`錆びた鎖` は取得時 HP +1 / 各フロア手数上限 -2。`棘の印` は取得時 HP +1 / 次に受けるダメージ +1、1回発動で消費。`血の契約` は取得時 HP +2 / 次に新しく得る報酬カードの使用回数 -1、最低1、報酬カード取得時に消費。`呪われた王冠` は新しく得る報酬カードの使用回数 +2 / 各フロア手数上限 -4。`黒曜の心臓` は取得時 HP +4 / 各フロア開始時 HP -1、最低1。`歪んだ砂時計` は各フロア手数上限 +6 / 拾得カードと報酬カードの使用回数増加量 -1、最低1。`赤い杯` は取得時 HP +6 / 以後受けるダメージ +1。`欲深い袋` は拾得カードの取得時使用回数 +2 / 新しく得る報酬カードの使用回数 -2、最低1。`割れた羅針盤` はクリア後の報酬候補 +1、最大4択 / 各フロア手数上限 -3。`重い鐘` は取得時 HP +2 / 各フロア最初の行動が2手分。`曇った鏡` は報酬候補 +1、最大4択 / レリック報酬候補に手早いクリアによる品質補正をかけない。`割れた靴` は取得時 HP +3 / レイ型移動報酬カードの使用回数 -1、最低1。`見張りの焼印` は取得時 HP +2 / 見張り・回転見張りから受けるダメージ +1。`巡回の鈴` は報酬候補 +1、最大4択 / 巡回兵から受けるダメージ +1。`追跡の匂い袋` は手数上限 +3 / 追跡兵から受けるダメージ +1。`流星の避雷針` は取得時 HP +3 / メテオと標的警告から受けるダメージ +1。`罠寄せの磁石` は新規報酬カード使用回数 +1 / ダメージ罠から受けるダメージ +1。`油染みの靴` は手数上限 +2 / 溶岩から受けるダメージ +1。`硝子の足環` は取得時 HP +2 / 落下ダメージ +1。`毒見の小瓶` は拾得カード使用回数 +1 / 毒ダメージ回数 +1。`鉄の足枷飾り` は取得時 HP +3 / 足枷中の行動が3手分。`愚者の仮面` は報酬候補 +1、最大4択 / 幻惑罠で追加手札喪失。`ほつれた記憶` は補助報酬カード使用回数 +1 / 手札喪失系罠で追加手札喪失。`湿った火口` は取得時 HP +2 / 暗闇視界半径 -1、最低1。`笑う扉` は報酬候補 +1、最大4択 / ワープマス移動時に手札を1つ喪失。`逆さ鍵` は鍵解錠後の報酬候補 +1、最大4択 / その階の手数上限 -2。`取り立て人` は報酬候補 +1、最大4択 / クリア報酬選択時に次階開始HP -1、最低1。`揺らぐ焚き火` は回復マス回復量 +2 / 回復マスで幻惑付与。
- 同一区間内では前フロアの階段位置を次フロアの開始位置にし、階段位置も複数パターンへ分散して周回時の固定感を減らす。
- ひび割れ床は区切り階や最終階など、通常遷移できる次階がない階には置かない。落下専用宝箱へつながるひび割れ床は、通常クリアに不要な任意リスクとして置き、落下先の小部屋から帰還ワープで通常ルートへ戻せる配置にする。

## 6. 永続成長

- 成長ポイントは成長塔カード内の「成長」で使用し、準備、報酬、危険回避、索敵、復帰の分岐網型スキルツリーから取得する。表示はまず系統ごとの役割カードを並べ、選んだ系統を50Fまでの縦タイムラインで読む形にする。「全体」表示では5系統の階層ノード図も確認できる。詳細説明、取得、オン/オフは選択中スキルの詳細欄で扱う。
- 成長ポイントは成長塔の5F刻み初回クリアで各1得る。区間終端の10F/20F/30F/40F/50Fを再クリアした場合も周回報酬として各1得られ、周回すれば最終的に全スキルを取得できる。
- 取得済みスキルはコストなしで個別にオン/オフでき、取得条件は取得時だけを見る。前提スキルをオフにしても、取得済みの後続スキルは単体で有効にできる。
- 準備系は区間開始カード、補給、防御補助、深層向け移動支度を扱う。報酬系は候補補完、カード温存、4択化、補助/遺物/強化候補の保証を扱う。危険回避系は罠・床崩落、敵、メテオ、深層向け保険を扱う。索敵系は床/報酬/敵/経路の見通しを扱う。復帰系は再挑戦、区間立て直し、将来の深層チェックポイント拡張を扱う。
- 一部の深層スキルは複数系統の取得済みスキルを前提にする。例として「深層補給術」は準備系と報酬系、「危険予報」は索敵系と危険回避系を前提にする。
- フロア間の HP 引き継ぎは通常どおり `DungeonRunState.carriedHP` を使い、永続成長で HP は増やさない。

## 7. 敵・床・ギミック

- 敵は `EnemyDefinition` で配置し、進行中は `EnemyState` として管理する。現行では番兵、巡回兵、見張り、回転見張り、追跡兵、メテオ兵を扱う。
- 敵本体の盤面マーカーは種類ごとに形を分ける。番兵は盾、巡回兵は巡回矢印、見張りは目、回転見張りは瞳孔部分の右回り/左回り矢印、追跡兵は追跡矢印、メテオ兵は落星印を使い、行動前にタイプを判別できるようにする。遊び方内の敵辞典にも同じ分類と読み方を載せる。
- 遊び方内の辞典は発見型にする。敵、マス、カード、レリック、呪い、イベントは、盤面、手札、床拾得、報酬候補、取得結果として表示された時点で発見済みにし、未発見項目は `？？？` として名称・見た目・効果を隠す。開発者向け診断メニューの全表示トグルがオンの場合だけ、発見状態に関係なく全詳細を見せる。
- 敵は種類を問わず、プレイヤーがそのマスを踏むと倒れる。レイ型、吹き飛ばし、ワープなどで通過したマスも対象になり、踏んだ敵はその手の敵ターンでは攻撃しない。
- 敵の危険範囲表示は敵本体マスを含めず、敵が攻撃できる周囲/射線/警告対象だけを塗る。敵本体マスは敵マーカーとして表示し、入ると撃破する対象として扱う。
- レイ型などの連続移動で途中マスが敵の危険範囲に入った場合、そのマスへ入った時点でダメージを受ける。途中で踏み潰した敵の危険範囲は以後消える。
- 凍結の呪文は移動せず1手使い、使用直後から合計3回分の敵ターンを止める。凍結中は危険範囲、着弾予告、次移動プレビューを表示せず、敵の攻撃範囲やメテオ予告に入ってもダメージを受けない。敵本体マーカーは残り、プレイヤーが敵マスを踏めば通常どおり撃破する。
- 障壁の呪文が有効な間も、危険範囲、着弾予告、次移動プレビュー、敵本体マーカーは通常どおり表示する。敵やメテオによるHPダメージは無効化され、既存の1回きり敵/メテオ軽減回数は消費しない。
- ダークネスの呪文が有効な階では、見張りと回転見張りの射線危険を表示せず、その射線上に入ってもダメージを受けない。敵本体マーカー、回転見張りの回転、その他の敵の危険範囲や着弾予告は通常どおり表示する。
- レール破壊の呪文が有効な階では、巡回兵の巡回レールと次移動プレビューを表示せず、敵ターンで巡回兵は移動しない。巡回兵の本体マーカー、隣接危険、隣接/接触ダメージは通常どおり残す。
- 敵同士は同じマスに重ならない。敵ターンでは配列順に移動を解決し、他の敵がいるマス、または同じ敵ターン内で先に予約された移動先へは進まずその場に残る。
- 巡回兵は巡回路全体を薄い灰色のレールとして常時表示し、現在地から次に進む方向は巡回兵アイコン内の向きで表示する。プレイヤーが巡回兵の現在地からの危険範囲に入った場合は、その敵は進まずに攻撃する。
- 回転見張りは4方向固定で、プレイヤーの1手ごとに右回りまたは左回りへ視線方向を切り替える。盤面の赤い射線は次の敵ターンに被弾する回転後の危険範囲として表示し、次の向きは敵アイコンの瞳孔部分にある矢印から読む。
- 追跡兵はプレイヤーの1手後に最短経路で1マス近づく。通行不可マス、崩落床、盤外は避け、到達不能なら動かない。表示する危険範囲は上下左右1マスで、敵本体マスは敵マーカーとして分ける。次に動くマスへの小さな矢印も常時表示する。プレイヤーが現在地からの危険範囲に入った場合は、追跡せずにその場で攻撃する。
- メテオ兵は移動せず、通行可能な床から制御ランダムに選んだ複数マスを着弾予告マスとして表示する。着弾予告マスは現在の危険範囲とは別扱いで、次の敵ターンにそのマスへいると1ダメージを受ける。現在地即被弾、通行不可マス、崩落床、敵本体マス、鍵/出口の同時封鎖は避ける。
- 暗闇フロアでも見張り/回転見張りのレーザー危険マスとメテオ着弾予告マスは視界外まで表示する。凍結中は既存通り危険範囲、着弾予告、次移動プレビューを消し、ダークネスの呪文が有効な階では見張り系レーザー表示も消す。
- `HazardDefinition.damageTrap` は見えている罠床として扱い、基本移動やカード移動で入った罠マスごとに HP を減らす。カード移動の途中マスも対象になる。
- `HazardDefinition.lavaTile` は見えている溶岩床として扱い、基本移動やカード移動で入った溶岩マスごとに HP を減らす。溶岩上で補助カードなどの移動しない行動を使った場合も、補助効果の解決後に HP を減らしてから敵ターンへ進む。
- 障壁の呪文が有効な間は、ダメージ罠、溶岩、毒、崩落穴のHPダメージも無効化する。足枷、手札喪失、沼による停止、床破壊、落下先への遷移など、HP減少以外の効果は無効化しない。
- `HazardDefinition.healingTile` は見えている回復マスとして扱い、基本移動やカード移動で初めて入った時に HP を 1 増やして消費する。HP 上限は設けず、カード移動の途中マス、吹き飛ばし経路、ワープ先も対象になる。成長塔中盤以降だけに少量配置する。
- `HazardDefinition.brittleFloor` は 1 回踏むとひび割れ、2 回目で崩落して穴として残る。崩落穴へ入るたびに HP を 1 減らし、報酬選択や成長ポイント付与を発生させず、短い落下演出の後に開始画面を挟まず前の階の同じマスへ落下する。前の階がない場合は失敗扱いにする。敵の移動、危険範囲、着弾予告は崩落穴を避ける。
- `TileEffect.swamp` は沼として扱い、移動系カードで入るとそのマスで止まる。沼の上では移動系カードを使えないが、基本移動と補助カードは通常どおり使える。
- `TileEffect.shackleTrap` は足枷罠として扱う。踏むとそのマスで停止し、次の階まで足枷状態になる。足枷状態では基本移動、移動カード、補助カードの全行動が 2 手分になり、敵ターンも最大 2 フェーズ進む。麻痺罠と重なっても敵ターンは 2 フェーズを上限とする。足枷は万能薬で解除できる。
- `TileEffect.poisonTrap` は毒罠として扱う。踏んだ手ではダメージを受けず、移動カードも止まらない。次の成功行動から毒カウントが進み、3 行動ごとに HP を 1 失う。毒ダメージは最大 3 回で、再度踏むと残り回数とカウントを初期値に戻す。毒は解毒薬または万能薬で解除できる。
- `TileEffect.illusionTrap` は幻惑罠として扱う。踏むとその階にいる間、移動カードの正体が `？` 表示になり、移動カードを押すと現在合法に使える移動カードと移動先からランダムに解決する。基本移動と補助カードは通常どおり使え、幻惑は万能薬または次フロア開始で解除される。
- `DungeonRules.exitLock` が設定されたフロアでは、指定された鍵マスを踏むまで出口/階段は施錠中として扱う。鍵は階段ロック解除専用で、取得後は床落ちカードと同じように盤面から消える。
- 成長塔と試練塔では `DungeonFloorDefinition.warpTilePairs` から `TileEffect.warp` を自動生成し、2点以上のワープ入口を循環ワープとして扱う。
- 試練塔の生成フロアは、移動不能マスだけで入口から出口までの上下左右経路が詰む盤面を生成しない。入口、出口、鍵、敵、罠、拾得カード、通行不可、ワープを重ねない。1Fから数階は軽めにし、以後は階数に応じて罠、敵、床割れ、ワープ、暗闇、状態異常床を段階的に増やす。罠・敵・手数制限によってカードを使わないと被弾や疲労が起きやすい階は許容する。
- 吹き飛ばしマスは `TileEffect.blast(direction:)` で上下左右いずれかの固定方向を持つ。踏むと矢印方向へ、盤外または通行不可マスに当たる直前まで移動し、通過したマスは踏破・敵撃破・罠・鍵・出口・拾得カードなど通常移動と同じ処理を行う。
- 手札喪失罠は `TileEffect.discardRandomHand`、`discardAllMoveCards`、`discardAllSupportCards`、`discardAllHands` で表す。踏むと移動後に、ランダム1枠、移動カード全枠、補助カード全枠、または全手札枠を失い、失った枠は即補充しない。塔の所持カード式では対象所持枠の残使用回数を0にする。複数の手札喪失罠を同じ移動で踏んだ場合は、全手札喪失を最優先し、移動カード喪失と補助カード喪失を両方踏んだ場合は両カテゴリを失う。ランダム1枠喪失はカテゴリ全喪失より軽い効果として扱う。
