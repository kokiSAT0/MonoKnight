# MonoKnight

SwiftUI と SpriteKit を用いたカード移動パズルのプロトタイプ。

## テストの実行方法

本リポジトリでは **Swift Package Manager** を利用して単体テストを管理しています。

```bash
# 依存関係の解決とテストの実行
swift test
```

上記コマンドをプロジェクトルートで実行すると、`Game` モジュールに対するテストが実行されます。
